# Proyecto Excursionista
# El objetivo de la aplicación consiste en determinar los elementos optimos para escalar un risco
de acuerdo al minimo de calorias y maximo peso, el resultado de la aplicacion debe ser indicar al usuario
el conjunto de elemento optimos

## Cómo ejecutar
Paso 1: El proyecto por defecto esta inicializado en la pagina de login, por lo que se deberan
introducir credenciales, las cuales son: usuario --admin y contraseña--123, si las credenciales no son 
correctas el sistema indicara que es invalido, de lo contrario mostrará la pagina de inicio

Paso 2: Al introducir usuario y contraseña correctos, dar clic en el boton Ingresar, esto lo redireccionara
a la pagina de Inicio, donde encontrara los campos para realizar el calculo del algoritmo

Paso 3: Introducir minimo de calorias y peso maximo, dar clic en el boton Calcular, el sistema
mostrará el resultado de los elementos optimos de acuero al peso maximo y minimo de calorias.

###Tecnologias usadas
- ASP.NET Web Forms
- C#
- Sql Server
- Bootstrap (Framework)
